# FlappyBirdClone
Simulation of popular bird game using sprite kit

### Gameplay
User must continuously tap screen to help bird fly between pipe obstables.  Each successive obstacle cleared (made through the pipes with out contact) adds a point to the users overall score.  Beware, the location of the opening of the obstacles will change.  Also contact with the ground will also end the game.  Good luck!
